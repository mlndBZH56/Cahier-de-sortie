This is an Android app in Kotlin using Jetpack Compose.

Architecture:
- MVVM
- Local database: Room
- Offline-first app

Main features:
- Rowing session logbook (cahier de sortie)
- Sessions created in 2 steps: start and end
- Boats, rowers, destinations
- Statistics and history

Rules:
- Keep UI simple for tablet use
- Use clean architecture
- Do not overcomplicate
